# PuzzleMaster

Authors: 
Matthew Gadke, Matt Graber, Logan Kindschi, Alex Marohl, Gage Winsand

A online puzzle game.
Similar to other match 4 type games like Bejeweled but it was an interesting challenge to build ourselves.

Link to the game. (Currently Non-functional)
https://puzzlemaster.gamer.gd/

Features:
A menu system that can pass information between pages.
A scoring system that tracks player data in a database and can display leaderboards.
Admin access pages that allow a user to modify, create, or delete logged scores.
A ramping difficulty of the game as you move to higher levels within a run.
